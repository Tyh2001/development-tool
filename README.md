# Vue Devtools

## 前言

Vue 浏览器调试工具

官网地址：https://devtools.vuejs.org

## 文件说明

[vue2]vue-devtools.zip：Vue2 调试工具

[vue3]vue-devtools.zip：Vue3 调试工具

## 安装方式

1. 在谷歌浏览器打开 chrome://extensions/
2. 开启【开发者模式】
3. 点击【加载已解压的扩展程序】
4. 选择下载的用解压工具解压的 vue-devtools 文件，即可
